### Initialization

- Clone the repository

```bash
git clone
```

- Move to the repository folder

```bash
cd Object-Oriented-Principles
```

- Open on vscode

```bash
code .
```

- Move to the Assignment1 folder (for example)

```bash
cd Assignment1
```

- Comnpile

```bash
javac edu/ucalgary/oop/*.java
```

java edu.ucalgary.oop.AroundTheWorld
